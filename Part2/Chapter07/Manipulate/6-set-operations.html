Open console to experiment with d3-array's functions

Operations that return sets

    d3.difference([3,6,9], [2,4,6,8,10])
    d3.difference([5,10], [2,4,6,8,10])
    d3.difference([2,4,6,8,10], [5,10], [3,6,9])
    d3.union([5,10], [3,6,9], [2,4,6,8,10])
    d3.intersection([2,4,6,8,10], [3,6,9])
    d3.intersection([5,10], [3,6,9])

Operations that return Boolean values

    d3.disjoint([5,10], [3,6,9])
    d3.superset([1,2,3,4,5,6], [2,4,6])
    d3.subset([3,6], [1,2,3,4,5,6])

Set operations with strings

    d3.difference("dinosaur", "pterosaur")
    d3.intersection("dinosaur", "pterosaur")

D3 versions of JavaScript functions that accept iterables

    d3.some('abc-def',  ch => ch >= 'a' && ch < 'z')
    d3.every('abc-def', ch => ch >= 'a' && ch < 'z')
